Each listed company is believed to use Lisp either for internal use in an industrial application or as the basis for an industrial product.

Research activities in universities are not generally listed here, nor are freely available tools, libraries and source code or Lisp consultants. There are also references for uses of functional languages in general.

To create an exhaustive list would be a huge task. This listing represents only those we have stumbled across. To add (or remove) a listing, send us mail.
