Each year the Association of Lisp Users (the ALU) has a meeting and conference called Lisp Users and Vendors (LUV). Sometimes, this is co-located with another exposition such as Object World or OOPSLA.

  • OOP, February 5, 1997, Munich Germany. Includes meeting of the German Lisp User Group.

  • ACM SIGPLAN Programming Language Design and Implementation (PLDI'97) June 15-18, 1997, Las Vegas.
  • Object World East '97.
