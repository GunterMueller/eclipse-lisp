Eclipse defined
Class STREAM, Class INPUT-STREAM, Class OUTPUT-STREAM

Class Precedence Lists:

stream, standard-object, t

input-stream, stream, standard-object, t

output-stream, stream, standard-object, t

Description:

The following classes must be used as superclasses of user-defined stream classes. They are not intended to be directly instantiated; they just provide places to hang default methods.

Stream is the base class for all Eclipse streams.

Input-stream is a superclass of all input streams.

Output-stream is a superclass of all output streams.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, open-stream-p close.

Notes:

These classes supplement the CLIM stream classes.

Eclipse has a very narrow definition of fundamental-stream. Some of the methods that CLIM specifies as being defined for fundamental-stream, etc, are actually implemented in Eclipse on stream, input-stream or output-stream.
