Eclipse defined
Function EXIT

Syntax:

exit &optional status condition =>|

Arguments and Values:

status---a generalized boolean. The default is false.

condition---a condition object, or NIL. The default is NIL.

Description:

The exit function transfers control to the most recently established applicable restart named EXIT. The status argument is passed to the restart.

If no such restart exists, or if that restart returns to the caller, exit exits the application by calling the operating system exit utility, with a non-zero status.

When condition is non-nil, only those restarts are considered that are either explicitly associated with that condition, or not associated with any condition; that is, the excluded restarts are those that are associated with a non-empty set of conditions of which the given condition is not an element. If condition is NIL, all restarts are considered.

Side Effects:

Exits the application.

Affected By:

The presence of an exit restart.

Exceptional Situations: None.

See Also:

"Lisp top-level", clLispTopLevel, exit (restart).

Notes: None.
