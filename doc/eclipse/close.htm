CLIM defined
Generic Function CLOSE

Syntax:

close stream &key abort => result

Arguments and Values:

stream---a stream.

abort---a generalized boolean. The default is false.

result---t if the stream was open at the time it was received as an argument, or implementation-dependent otherwise.

Description:

The existing ANSI function close is redefined to be a generic function, but otherwise behaves the same.

Methods:

Primary Method close (stream stream)
Sets a flag used by open-stream-p.

Side Effects: Closes the stream.

Affected By: Previous calls to close.

Exceptional Situations: None.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes, open-stream-p.

Notes:

CLIM defines the default method on fundamental-input-stream rather than on stream.

Note that ANSI specifies that:

The effect of close on a constructed stream is to close the argument stream only. There is no effect on the constituents of composite streams.
This means, for example, that closing an encapsulating-stream does not close its encapsulated stream.
