CLIM defined
Function DEALLOCATE-RESOURCE

Syntax:

deallocate-resource name object => object

Arguments and Values:

name---a symbol naming a resource.

object---an element of the resource.

Description:

Deallocate-resource returns the object to the resource named name.

Side Effects:

Can effect the values of the arguments given to function by map-resource.

Affected By:

The resource named name.

Exceptional Situations: None.

See Also:

allocate-resource, clear-resource, defresource, map-resource, using-resource.

Notes: None.
