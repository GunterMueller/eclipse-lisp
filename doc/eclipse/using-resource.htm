CLIM defined
Macro USING-RESOURCE

Syntax:

using-resource (variable name &rest arguments) &body forms => values

Arguments and Values:

variable---a symbol naming a variable that will be bound to an element of the resource named by name.

name---a symbol naming a resource.

arguments---arguments matching the parameters of the corresponding defresource.

values---the values returned by forms.

Description:

Using-resource evaluates the forms within a context in which variable is bound to an object allocated from the resource named by name, using the values given by arguments. The arguments (if any) are evaluated, but name and variable are not.

After the forms have been been evaluated, using-resource returns the object in variable back to the resource. If some form in forms sets variable to NIL, the object will not be returned to the resource. Otherwise, forms should not change the value of variable,

Side Effects:

Can effect the values of the arguments given to function by map-resource.

Affected By:

The resource named name, clear-resource.

Exceptional Situations: None.

See Also:

allocate-resource clear-resource, deallocate-resource, defresource, map-resource.

Notes:

CLIM does not define the value returned by using-resource. The return of the values returned by forms is Eclipse-defined behavior.
