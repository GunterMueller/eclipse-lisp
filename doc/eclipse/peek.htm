CLIM defined
Generic Function STREAM-PEEK-CHAR

Syntax:

stream-peek-char stream => result

Arguments and Values:

stream---a character input stream.

char---a a character or the symbol :EOF.

Description:

Stream-peek-char is used by the ANSI defined function peek-char and returns either a character, or the symbol :EOF if stream is at end of file. The character returned is not removed from the input buffer of stream, and will be returned again on the next call to stream-peek-char, stream-read-char, or stream-read-char-no-hang.

Methods:

Primary Method stream-peek-char (stream fundamental-character-input-stream)
Calls stream-read-char and stream-unread-char.

Primary Method stream-peek-char (stream encapsulating-stream)
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.
