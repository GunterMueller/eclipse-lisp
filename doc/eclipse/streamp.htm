Eclipse defined
Generic Function STREAMP

Syntax:

streamp stream => result

Arguments and Values:

stream---a stream.

result---a generalized boolean.

Description:

The existing ANSI function streamp is redefined to be a generic function, but otherwise behaves the same.

Methods:

Primary Method streamp (stream stream)
Returns true.

Primary Method streamp (stream t)
Returns false.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes, close.

Notes:

CLIM specifies a default method only for fundamental-stream rather than on stream.
