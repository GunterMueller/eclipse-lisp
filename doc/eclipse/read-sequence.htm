Generic Function STREAM-READ-SEQUENCE, STREAM-WRITE-SEQUENCE

Syntax:

stream-read-sequence input-stream sequence &key start end => position

stream-write-sequence output-stream sequence &key start end => sequence

Arguments and Values:

input-stream---an input stream of element-type.

output-stream---an output stream of element-type.

sequence---a a sequence of element-type.

start, end---a bounding index designators of sequence. The defaults for start and end are 0 and NIL, respectively.

position---a an integer greater than or equal to zero, and less than or equal to the length of the sequence.

Description:

Stream-read-sequence is used by the ANSI defined function read-sequence and returns the index of the first element of sequence that was not updated, which might be less than end because the end of file was reached.

Stream-write-sequence is used by the ANSI defined function write-sequence and returns sequence.

Primary Method stream-read-sequence (stream input-stream) (sequence sequence) &key start end
Primary Method stream-read-sequence (stream output-stream) (sequence sequence) &key start end
Signals type-error. These provide the correct ANSI defined behavior when read-sequence or write-sequence is given an input-stream or output-stream that has a different element-type than sequence. It is up to the programmer to provide applicable overriding methods for correctly matching arguments.
