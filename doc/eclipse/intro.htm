  • Eclipse-specific Glossary
The Eclipse documentation assumes the reader is familiar with Common Lisp and its documentation. Throughout the Eclipse documentation, references to the ANSI standard, the standard, Common Lisp, or ANSI Common Lisp, refer to "ANSI X3.226:1994 American National Standard for Programming Language Common LISP".
