CLIM defined
Generic Function STREAM-UNREAD-CHAR

Syntax:

stream-unread-char stream char => char

Arguments and Values:

stream---a character input stream.

char---a a character.

Description:

Stream-unread-char is used by the ANSI defined function unread-char. The char argument must be the most recently read character from stream (eg. by stream-read-char or stream-read-char-no-hang) and is inserted back into the input buffer of stream.

Methods:

Primary Method stream-unread-char (stream encapsulating-stream)
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes, stream-read-char, stream-read-char-no-hang, stream-peek-char, stream-listen-char.

Notes: None.
