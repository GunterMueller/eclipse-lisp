CLIM defined
Function CLEAR-RESOURCE

Syntax:

clear-resource name => implementation-dependent

Arguments and Values:

name---a symbol naming a resource.

Description:

Clear-resource clears the resource named name, that is, it removes all of the resourced objects from the resource.

Side Effects:

Can effect the values of the arguments given to function by map-resource.

Affected By: None.

Exceptional Situations: None.

See Also:

allocate-resource, deallocate-resource, defresource, map-resource, using-resource.

Notes: None.
