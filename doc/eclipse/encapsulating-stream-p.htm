Eclipse defined
Generic Function ENCAPSULATING-STREAM-P

Syntax:

encapsulating-stream-p stream => result

Arguments and Values:

object---any object.

result---a generalized boolean.

Description:

Returns true if the object is an encapsulating-stream, otherwise returns false.

Methods:

Primary Method encapsulating-stream-p (stream encapsulating-stream)
Returns true.

Primary Method input-stream-p (stream t)
Returns false.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.

See Also:

"Extensions to Common Lisp", encapsulating-stream.

Notes: None.
