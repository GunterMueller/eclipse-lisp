CLIM defined
Generic Function OPEN-STREAM-P

Syntax:

open-stream-p stream => result

Arguments and Values:

stream---a stream.

result---a generalized boolean.

Description:

The existing ANSI function open-stream-p is redefined to be a generic function, but otherwise behaves the same.

Methods:

Primary Method open-stream-p (stream stream)
Returns true if close has not been called on the stream.

Primary Method open-stream-p (stream t)
Signals type-error.

Side Effects: None.

Affected By: Previous calls to close.

Exceptional Situations:

Signals type-error if stream is not a stream.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes, close.

Notes:

CLIM defines the default method on fundamental-input-stream rather than on stream.
