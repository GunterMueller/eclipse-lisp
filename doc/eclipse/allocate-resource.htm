CLIM defined
Function ALLOCATE-RESOURCE

Syntax:

allocate-resource name &rest arguments => object

Arguments and Values:

name---a symbol naming a resource.

arguments---arguments matching the parameters of the corresponding defresource.

object---an element of the resource.

Description:

Allocate-resource allocates an object from the resource named name, using the values given by arguments.

Side Effects:

Can effect the values of the arguments given to function by map-resource.

Affected By:

The resource named name, clear-resource.

Exceptional Situations: None.

See Also:

clear-resource, deallocate-resource, defresource, map-resource, using-resource.

Notes: None.
