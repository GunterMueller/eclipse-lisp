CLIM defined
Generic Function STREAM-WRITE-STRING

Syntax:

stream-write-string stream string &key start end => string

Arguments and Values:

stream---an character output stream.

string---a a sequence.

start, end---a bounding index designators of sequence. The defaults for start and end are 0 and NIL, respectively.

Description:

Stream-write-string is used by the ANSI defined function write-string.

Methods:

Primary Method stream-write-string (stream fundamental-character-output-stream) (string string) &key start end
Calls stream-write-sequence

Primary Method stream-read-string (stream encapsulating-stream) (string string) &key start end
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes, stream-write-sequence, stream-write-char.

Notes:

CLIM requires implementations only to define a default method for write-sequence that iterates over string, repeatedly calling stream-write-char. Instead, the default method in Eclipse calls write-sequence (which is not defined at all by CLIM) which does the iteration.
