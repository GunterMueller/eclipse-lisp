Eclipse defined
Variable *DEFAULT-ELEMENT-TYPE*

Value Type: a character type-specifier.

Initial Value: CHARACTER.

Description:

The default value for the element-type argument to stream creation operations such as make-string-output-stream and open.

Affected By: None.

See Also:

"Characters" and init.lisp.

Notes:

The ANSI standard does not define this variable, and the default value for element-type is specified as being CHARACTER, which Eclipse uses as the initial value for this variable. Thus the default behavior is as per the ANSI standard.

The vast majority of Common Lisp applications only use BASE-STRING, particularly those that were developed on other implementations in which BASE-STRING is equivalent to STRING. In such applications, using arrays of EXTENDED-CHAR rather than BASE-CHAR is wasteful of both time and space.
