Eclipse defined
Function MACHINE-ID

Syntax:

machine-id => id

Arguments and Values:

id---an integer.

Description:

Machine-id returns the machine address on a network.

Side Effects: None.

Affected By:

The operating system.

Exceptional Situations: None.

See Also:

Machine-instance in the ANSI standard, *machine-version*

Notes:

The exact value returned by machine-id is implementation-dependent, but is intended to be unique across all machines. It may use, for example, the operating system's gethostid, getpitimer, or unmame
