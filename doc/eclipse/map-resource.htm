CLIM defined
Function MAP-RESOURCE

Syntax:

map-resource function name => NIL

Arguments and Values:

name---a symbol naming a resource.

function---a designator for a function of three arguments: an object from the resource, in-use-p, and name.

in-use-p---a boolean that is true if the object is in use (i.e. allocated) and false if it is free.

Description:

Map-resource calls function once on each object in the resource named name.

Function has dynamic-extent.

Side Effects:

The resource named name,

Affected By:

The resource named name, allocate-resource, deallocate-resource, and clear-resource

Exceptional Situations: None.

See Also:

allocate-resource, clear-resource, deallocate-resource, defresource, using-resource.

Notes: None.
