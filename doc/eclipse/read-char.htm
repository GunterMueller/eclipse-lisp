CLIM defined
Generic Function STREAM-READ-CHAR, STREAM-READ-BYTE, STREAM-WRITE-CHAR, STREAM-WRITE-BYTE

Syntax:

stream-read-char character-input-stream => char-result

stream-read-byte binary-input-stream => integer-result

stream-write-char character-output-stream char => char

stream-write-byte binary-output-stream integer => integer

Arguments and Values:

character-input-stream---a character input stream.

binary-input-stream---a binary input stream.

character-output-stream---a character output stream.

binary-output-stream---a binary output stream.

char---a a character.

char-result---a a character or the symbol :EOF.

integer---a an integer.

integer-result---a an integer or the symbol :EOF.

Description:

Stream-read-char is used by the ANSI defined function read-char and returns either a character, or the symbol :EOF if character-input-stream is at end of file.

Stream-read-byte is used by the ANSI defined function read-byte and returns either an integer, or the symbol :EOF if binary-input-stream is at end of file.

Stream-write-char is used by the ANSI defined function write-char. It writes char to the character-output-stream and returns char as the result.

Stream-write-byte is used by the ANSI defined function write-byte. It writes integer to the binary-output-stream and returns integer as the result.

Methods:

Primary Method stream-read-char (stream encapsulating-stream)
Primary Method stream-read-byte (stream encapsulating-stream)
Primary Method stream-write-char (stream encapsulating-stream) (char character)
Primary Method stream-write-byte (stream encapsulating-stream) (integer integer)
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.
