CLIM defined
Generic Function STREAM-CLEAR-INPUT, STREAM-CLEAR-OUTPUT, STREAM-FORCE-OUTPUT, STREAM-FINISH-OUTPUT

Syntax:

stream-clear-input input-stream => NIL

stream-clear-output output-stream => NIL

stream-force-output output-stream => NIL

stream-finish-output output-stream => NIL

Arguments and Values:

input-stream---an input stream.

output-stream---an output stream.

Description:

These generic functions provide the same behavior as the corresponding ANSI functions clear-input, clear-output, force-output, finish-output. They differ from ANSI functions in that the stream argument is not optional and must be a stream of the appropriate type. I.E it cannot be T, or NIL.

Methods:

Primary Method stream-clear-input (stream fundamental-input-stream)
Primary Method stream-clear-output (stream fundamental-output-stream)
Primary Method stream-force-output (stream fundamental-output-stream)
Primary Method stream-finish-output (stream fundamental-input-stream)
Does nothing.

Primary Method stream-clear-input (stream encapsulating-stream)
Primary Method stream-clear-output (stream encapsulating-stream)
Primary Method stream-force-output (stream encapsulating-stream)
Primary Method stream-finish-output (stream encapsulating-stream)
Calls the same generic function on the encapsulating-stream-stream of stream.

Primary Method stream-clear-input (stream t)
Primary Method stream-clear-output (stream t)
Primary Method stream-force-output (stream t)
Primary Method stream-finish-output (stream t)
Signals type-error.

Side Effects: None.

Affected By: None.

Exceptional Situations:

Signals type-error if stream is not a stream.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes.

Notes:

CLIM specifies a default method only for fundamental-character-input-stream and fundamental-character-output-stream (as appropriate) rather than on fundamental-input-stream and fundamental-output-stream.
