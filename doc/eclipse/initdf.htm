Eclipse defined
C Function clInitD

Syntax:

clInitD => void

Description:

Initializes all data used by, and available from, the Eclipse development library (2).

clInit must be called before clInitD.

Side Effects:

After clInitD returns, all Lisp external and internal symbols are interned, bound and fbound appropriately, and have correct property lists. However, clInitD is not required to reinitialize any such data if called repeatedly, or after the system has already been initialized in some other way.

Affected By: None.

Exceptional Situations: None.

See Also:

Initializing the Eclipse Library, clInit, Lisp Top-Level.

Notes:

Any stand-alone (2) dynamic application must be initialized by calling clInitD.
