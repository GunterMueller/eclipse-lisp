Eclipse C: Initializing the Eclipse Library

The following functions are among those defined in the Eclipse libraries:
  • clInit -- Initializes all data used by the Eclipse run time library.
  • clInitD -- Initializes all data used by the Eclipse development library. clInit must be called before clInitD is called.
The functions above initialize the data used by various utilities in the Eclipse library. This includes:
  • The interning of symbols.
  • The creation of compile-file literal data (i.e. any Lisp data used by the library).
  • Assignment of symbol properties, such as symbol-function for symbols naming functions, or symbol-value for symbols naming special variables.
The sources for these functions are provided.

Later releases may break these down into finer grained pieces, but the two functions above will always be available.

See also, "The Eclipse Libraries".
