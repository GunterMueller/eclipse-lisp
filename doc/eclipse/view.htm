Eclipse Introduction: Viewing the Documentation

The Eclipse documentation is provided as HTML files which may be viewed using any WWW browser.

Using http/UNIX pathnames, the files are in eclipse-directory/doc/eclipse/ and the initial page is eclipse-directory/doc/eclipse/index.htm, where eclipse-directory is the name of the directory in which Eclipse was installed.

To read the documentation, instruct your WWW browser to "open file" or "read file" and select the file doc/eclipse/index.htm. Alternatively, you can use the "open location" or "goto location" command and specify a URL of 

  file:eclipse-directory/doc/eclipse/index.htm
where eclipse-directory is the location in which the distribution was placed, beginning with the root slash (/).

For your convenience, a local copy of the material from the Association of Lisp Users' web-site is also included with the Eclipse documentation. The files are in eclipse-directory/doc/alu/ and the initial page is eclipse-directory/doc/alu/index.htm.

Pages from the documentation may be printed using your browser's print facility.
