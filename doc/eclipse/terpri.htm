CLIM defined
Generic Function STREAM-TERPRI, STREAM-FRESH-LINE

Syntax:

stream-terpri stream => NIL

stream-fresh-line stream => fresh-line-p

Arguments and Values:

stream---a character output stream.

fresh-line-p---a a generalized boolean.

Description:

Stream-terpri is used by the ANSI defined function terpri. It writes #\newline to stream and return nil.

Stream-fresh-line is used by the ANSI defined function fresh-line. It writes #\newline to stream and return true only if stream is not at the beginning of the line. Otherwise it just returns false.

Methods:

Primary Method stream-terpri (stream fundamental-character-output-stream)
Writes #\newline to stream using stream-write-char.

Primary Method stream-fresh-line (stream fundamental-character-output-stream)
Calls stream-terpri and returns true iff stream-start-line-p returns true.

Primary Method stream-terpri (stream encapsulating-stream)
Primary Method stream-fresh-line (stream encapsulating-stream)
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.
