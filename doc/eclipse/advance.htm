CLIM defined
Generic Function STREAM-ADVANCE-TO-COLUMN

Syntax:

stream-advance-to-column stream column => result

Arguments and Values:

stream---a character output stream.

column---an integer.

result---a a generalized boolean.

Description:

Stream-advance-to-column writes enough blank space on stream so that the next character will be written at the position specified by column. Returns true if the the operation is successful, or false if it is not supported for stream. This function is used in the implementation of pprint and the format ~T directive.

Methods:

Primary Method stream-start-line-p (stream fundamental-character-output-stream)
Returns false if stream-line-column returns NIL. Otherwise, it repeatedly calls stream-write-char with a #\space character and returns true.

Primary Method stream-advance-to-column (stream encapsulating-stream)
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.
