CLIM defined
Generic Function STREAM-ELEMENT-TYPE

Syntax:

stream-element-type stream => element-type

Arguments and Values:

stream---a stream.

element-type---a type-specifier

Description:

Stream-element-type is a generic-function that implements the ANSI specification of stream-element-type.

Methods:

Primary Method stream-element-type (stream fundamental-stream)
Returns the value given as the element-type initarg when stream was instantiated.

Primary Method stream-element-type (stream encapsulating-stream)
Returns the stream-element-type of the encapsulating-stream-stream of stream.

Primary Method stream-element-type (stream t)
Signals type-error.

Side Effects: None.

Affected By: The initargs given to the stream when it was created.

Exceptional Situations:

Signals type-error if stream is not a stream.

See Also:

"Extensions to Common Lisp", "CLIM: Stream Classes", "Eclipse: Stream Classes".

Notes:

CLIM defines a method only on fundamental-character-stream, and specifies that it always returns character. In Eclipse, it may return a subtype of character.
