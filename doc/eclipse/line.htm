CLIM defined
Generic Function STREAM-READ-LINE

Syntax:

stream-read-line stream => line, missing-newline-p

Arguments and Values:

stream---a character input stream.

line---a a string or the symbol :EOF.

missing-newline-p---a generalized boolean.

Description:

Stream-read-line is used by the ANSI defined function read-line.

The primary value, line, is the line that is read, represented as a string (without the trailing newline, if any). If the end of file for stream is reached before any characters are read, the symbol :EOF is returned as the line.

The secondary value, missing-newline-p, is false if the line was terminated by a newline, or true if the line was terminated by the end of file for stream.

Methods:

Primary Method stream-read-line (stream fundamental-character-input-stream)
Calls stream-read-char repeatedly.

Primary Method stream-read-line (stream encapsulating-stream)
Calls the same generic-function on the encapsulating-stream-stream of stream.

Side Effects: None.

Affected By: None.

Exceptional Situations: None.
