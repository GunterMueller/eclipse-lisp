Eclipse defined
Generic Function INPUT-STREAM-P, OUTPUT-STREAM-P

Syntax:

input-stream-p stream => result

input-stream-p stream => result

Arguments and Values:

stream---a stream.

result---a generalized boolean.

Description:

The existing ANSI functions input-stream-p and output-stream-p is redefined to be a generic function, but otherwise behaves the same.

Methods:

Primary Method input-stream-p (stream input-stream)
Primary Method output-stream-p (stream output-stream)
Returns true.

Primary Method input-stream-p (stream encapsulating-stream)
Primary Method output-stream-p (stream encapsulating-stream)
Calls the same generic function on the encapsulating-stream-stream of stream.

Primary Method input-stream-p (stream stream)
Primary Method output-stream-p (stream stream)
Returns false.

Primary Method input-stream-p (stream t)
Primary Method output-stream-p (stream t)
Signals type-error.

Side Effects: None.

Affected By: None.

Exceptional Situations:

Signals type-error if stream is not a stream.

See Also:

"Extensions to Common Lisp", CLIM Stream Classes, Eclipse Stream Classes.

Notes:

CLIM specifies a default method only for fundamental-stream rather than on stream.
