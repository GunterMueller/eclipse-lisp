Compound Type Specifier C:FUNCTION

Compound Type Specifier Syntax:

c:function arg-typespec value-typespec

Compound Type Specifier Arguments:

arg-typespec---a list of zero or more type specifiers.

value-typespec---a type-specifier.

Compound Type Specifier Description:

The list form of the c:function type-specifier can be used only for declaration and not for discrimination. Every element of this type is a function that accepts arguments of the types specified by the argj-types and returns values that are members of the types specified by value-type.

When a call to a function that has been declared with a c:function type specifier is translated to C using compile-file, the generated C code will use calling conventions appropriate for a call to a C function.

See Also:

The function type specifier in the Common Lisp standard, "Delivering Applications", "Lisp Function Calls in C", c:macro.

Notes:

Unlike the ordinary Lisp function compound type specifier, the &optional, &rest, &key, and &allow-other-keys markers cannot appear in the list of argument types.

It may be useful to use symbols in the C package to identity type specifiers for use in this compound type specifier.
