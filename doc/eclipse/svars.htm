Eclipse behavior for
Variable *DEBUG-IO*, *ERROR-OUTPUT*, *QUERY-IO*, *STANDARD-INPUT*, *STANDARD-OUTPUT*, *TRACE-OUTPUT*

Initial Values:

*Standard-input* is initially bound to a file-stream equivalent to the C stream stdin.

*Standard-output* and *trace-output* are initially bound to a file-stream equivalent to the C stream stdout.

*Error-output* is initially bound to a file-stream equivalent to the C stream stderr.

*Terminal-io* is initially bound to a two-way-stream made from the initial value of *standard-input* and the initial value of *standard-output*.

*Query-io* and *debug-io* are both initially bound to synonym-streams to *terminal-io*.

Description: None.

Affected By: Dribble.

See Also:

The Streams and the Miscellaneous: Dribble sections of "Eclipse-Defined ANSI Features".
