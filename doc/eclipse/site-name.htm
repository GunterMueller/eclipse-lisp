Eclipse defined
Variable *MACHINE-VERSION*, *SHORT-SITE-NAME*, *LONG-SITE-NAME*

Value Type:

a string or NIL.

Initial Values:

NIL

Description:

The function short-site-name returns the value of the variable *short-site-name*.

The function long-site-name returns the value of *long-site-name* if it is non-null, otherwise it returns *short-site-name*.

The function machine-version returns the value of the variable *machine-version*. However, if *machine-version* is NIL and the system can obtain the machine version from the operating system, machine-name returns that value rather than NIL.

Affected By:

Usually set in init.lisp.

See Also:

"Site and Other Initial Definitions" and "init.lisp", machine-id.

Notes: None.
