Eclipse defined
Variable *READ-EVAL-PRINT-HELP*

Value Type:

a string.

Initial Values:

A string that describes the operation of the read-eval-print loop.

Description:

The value of this string is printed by the read-eval-print loop in response to a request for help.

Affected By: Nothing.

See Also:

"The Real-Eval-Print Loop And Debugger".

Notes: None.
